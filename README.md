# Landing Page Nutricionista Alissia Rachel

Uma landing page profissional e responsiva desenvolvida para apresentação de serviços de nutrição, focado em conversão e agendamento de consultas.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as ferramentas mais modernas e robustas do ecossistema front-end:

*   **[Next.js](https://nextjs.org/)** (v16) - Framework React da Vercel.
*   **[React](https://react.dev/)** (v19) - Biblioteca principal para construção da interface.
*   **[Tailwind CSS](https://tailwindcss.com/)** (v4) - Estilização utilitária super rápida e moderna.
*   **Componentes:** Radix UI, Embla Carousel, Lucide React (Ícones).
*   **Formulários:** React Hook Form validado com Zod.
*   **Linguagem:** TypeScript para maior segurança e previsibilidade no código.

## 📦 Como rodar o projeto localmente

1. **Clone este repositório:**
```bash
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
```

2. **Acesse a pasta do projeto:**
```bash
cd Landing-Page-Nutricionista-Alissia-Rachel
```

3. **Instale as dependências:**
```bash
npm install
```

4. **Inicie o servidor de desenvolvimento:**
```bash
npm run dev
```

5. **Abra no navegador:**
O site estará rodando em [http://localhost:3000](http://localhost:3000).

## 🛠️ Scripts Disponíveis

No terminal, você pode rodar os seguintes comandos:

*   `npm run dev`: Inicia o ambiente de desenvolvimento.
*   `npm run build`: Cria a versão otimizada para produção.
*   `npm start`: Roda o servidor com a versão de produção (precisa rodar o build antes).
*   `npm run lint`: Checa e ajuda a arrumar erros no código usando o ESLint.

## 💻 Estrutura do Projeto

*   `/app`: Contém as páginas e rotas do Next.js (App Router).
*   `/components`: Componentes reutilizáveis da interface (botões, cards, seções).
*   `/public`: Imagens, fontes e outros arquivos estáticos.
*   `/styles`: Configurações de estilo globais (CSS).
*   `/lib` & `/hooks`: Funções utilitárias e hooks customizados do React.
